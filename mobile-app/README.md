# 📱 RealityShield Mobile

A premium mobile application built with Expo for capturing and verifying secure photos.

## ✨ Features
- **Secure Camera**: Captures photos directly to prevent manipulation.
- **GPS Integration**: Attaches geographic proofs to the blockchain record.
- **Identity Vault**: View your history of secured assets.
- **Real-time Verification**: Instant feedback on image authenticity.

## 🛠️ Setup & Running
1. `npm install`
2. Configure `./.env` with your `EXPO_PUBLIC_API_URL`.
3. `npx expo start`
4. Use the **Expo Go** app on your phone to scan the QR code.

## 🎨 UI/UX
Designed with a sleek **Dark Mode** and **Glassmorphism** for a premium cryptographic feel.

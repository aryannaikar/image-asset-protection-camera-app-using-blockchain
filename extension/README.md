# 🌐 RealityShield Extension

A Chrome extension to verify images directly from your browser.

## 🛠️ Features
- **Context Menu Integration**: Right-click any image to verify its authenticity.
- **Visual Overlays**: Displays status badges directly on the webpage.
- **Vault Connection**: Links to the backend verification service.

## 🛠️ Installation
1. Open Chrome and go to `chrome://extensions/`.
2. Toggle **Developer mode** on.
3. Click **Load unpacked**.
4. Select this `extension` folder.

## ⚙️ Configuration
The extension connects to the backend at `http://127.0.0.1:5000` by default. This can be modified in `background.js`.
